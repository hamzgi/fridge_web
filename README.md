# 냉장고 재료 관리 웹앱 (Fridge Manager)

간단한 웹앱으로, 냉장고 안 식재료와 유통기한을 관리할 수 있습니다.  
Firebase Firestore를 사용하여 데이터 저장, Firebase Hosting을 통해 배포했습니다.  

---

## 🔹 주요 기능

- **식재료 추가**: 이름과 유통기한 입력 후 목록에 추가  
- **식재료 목록 확인**: 현재 냉장고에 있는 재료와 유통기한 확인 가능  
- **식재료 삭제**: 사용한 재료나 필요 없는 항목 삭제 가능  
- **웹앱 배포**: Firebase Hosting을 통해 언제 어디서든 접속 가능  
- **모바일 지원**: 모바일 브라우저에서도 사용 가능  

---

## 🔹 사용 기술

- **Frontend**: HTML, CSS, JavaScript  
- **Backend / Database**: Firebase Firestore  
- **Hosting**: Firebase Hosting
